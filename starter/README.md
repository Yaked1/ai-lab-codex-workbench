# Workbench Starter

Copy this directory into a new repository when you want the smallest coherent
slice of the workbench. It has no required network dependency and contains no
private paths.

## Contents

- `task-template.md`: bounded agent work order.
- `safety-rules.md`: default permission and data boundaries.
- `evaluation.md`: acceptance and review rubric.
- `example/`: one filled task and expected report.

## Use

1. Copy `starter/` into the target repository.
2. Fill `task-template.md` with exact files, exclusions, and checks.
3. Keep `safety-rules.md` unless the repository has stricter rules.
4. Score the result with `evaluation.md` before accepting it.
