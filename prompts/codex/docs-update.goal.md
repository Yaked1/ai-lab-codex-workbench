# Codex Prompt: Documentation Update

```text
/goal
Objective:
Improve or add documentation for:
[Topic.]

Success criteria:
- Documentation is clear for a beginner.
- Windows PowerShell commands are used where setup commands are needed.
- Hardware limits are stated if relevant.
- No unsupported model claims are invented.
- Links are kept accurate when included.
- Safe autofix check passes.

Safety boundaries:
- Do not edit code unless necessary.
- Do not add heavy dependencies.
- Do not touch secrets.
- Do not rewrite unrelated docs.

Final response:
- Summary
- Files changed
- Commands run
- Remaining risks or unclear facts
```
