# OpenCode

OpenCode is an open-source AI coding agent available through terminal, desktop, or IDE-oriented workflows depending on the current release. It can connect to different model providers and is useful for users who want an open-source agent workflow.

## Best For

- Terminal-first coding sessions.
- Read-only repository exploration.
- Provider-flexible experiments.
- Comparing open-source agent behavior with commercial tools.

## Beginner Fit

Medium. It is friendly for users comfortable with the terminal, but model/provider setup and permissions should be explained clearly.

## Windows Fit

Verify the current Windows installation path and desktop support before teaching setup. Keep first tasks read-only if the environment is new.

## Surface

CLI, desktop, IDE, and hybrid surfaces may be available depending on release.

## Main Risks

- Misconfigured provider credentials.
- Unexpected terminal command execution.
- Cost surprises when switching providers.
- Assuming every install method works the same on Windows.

## Best First Task

Ask OpenCode to summarize the repository structure without editing files.
