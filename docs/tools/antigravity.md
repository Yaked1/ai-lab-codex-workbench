# Google Antigravity

Google Antigravity is an agentic development platform for launching, monitoring, and coordinating AI agents across development work. Treat it as a fast-changing tool and verify current platform support before publishing setup instructions.

## Best For

- Agent-first project planning.
- Structured artifacts.
- Parallel or coordinated agent work when the task is well bounded.
- Comparing agent outputs before choosing an implementation.

## Beginner Fit

Medium. The agent-first model can be powerful, but beginners should start with planning artifacts and documentation tasks before allowing broad code edits.

## Windows Fit

Verify the current Windows support, installer, CLI, and IDE instructions in the official docs. Prefer browser or cloud paths for limited laptops when available.

## Surface

Antigravity may involve IDE, CLI, cloud, or hybrid surfaces depending on the current product and version.

## Main Risks

- Parallel agents drifting into unrelated files.
- Confusing generated artifacts with verified results.
- Preview or rapidly changing product behavior.
- Unclear permissions when agents can run tools.

## Best First Task

Ask Antigravity to create a plan artifact for a documentation cleanup. Do not permit code edits until the plan is reviewed.
