# First Success in 15 Minutes

Use the repository's
[first reviewed agent task](../../examples/first-reviewed-agent-task/README.md).
It gives you a bounded input, a ready work order, an expected report, and a
local checker.

## Outcome

You finish with:

- `examples/first-reviewed-agent-task/report.md`;
- a visible `PASS` result from `check_report.py`;
- a concrete example of objective, scope, output contract, and verification.

## Next route

After the check passes, choose the one-hour reliable-agent track in
[Prompting OS](README.md) or use the maintainer reference only when you are
working on the repository itself.
