## Confirmed

- GPT-5.6 launched generally on 2026-07-09 as Sol, Terra, and Luna. Standard
  ChatGPT exposes Sol through Medium, High, and Extra High; Terra and Luna are
  available in Work, Codex, and the API subject to plan.
- GPT-5.6 API base prices are Sol $5/$30, Terra $2.50/$15, and Luna $1/$6 per
  million input/output tokens. Inputs above 272K use 2x input and 1.5x output
  rates for the full request.
- The local Codex catalog exposed Sol low/medium/high/xhigh/max/ultra, Terra
  low/medium/high/xhigh/max/ultra, and Luna low/medium/high/xhigh/max.
- Fable 5 API effort values are low/medium/high/xhigh/max. Ultracode is a
  Claude Code orchestration mode, not an API effort above max.
- Fable 5 promotional included access was extended through 2026-07-12 at
  11:59:59 PM PT for eligible paid plans, then moves to usage credits.
- Fable 5 standard pricing is $10/$50 per million tokens across the full 1M
  context window, including inputs above 200K.
- Grok 4.5 is the default Grok Build model, supports low/medium/high reasoning,
  and costs $2/$6 per million base input/output tokens.
- Gemini 3.5 Flash API supports minimal/low/medium/high thinking; Gemini Apps
  uses Standard/Extended labels where available.
- Gemini 3.5 Live Translate API ID is `gemini-3.5-live-translate-preview`; it is
  audio-only translation with 70+ languages and launched in public preview in
  the Live API and AI Studio.

## Vendor claims

- OpenAI says GPT-Live uses a continuous full-duplex architecture and can
  delegate deeper work while maintaining the spoken interaction.

## Independent evidence

- Artificial Analysis scores Fable Max 60 and Sol Max 59 on its Intelligence
  Index v4.1.
- Artificial Analysis scores Sol Max in Codex 80 and Fable Max with fallback in
  Claude Code 77 on its Coding Agent Index v1.1.
- Arena's 2026-07-10 text snapshot gives Fable the stronger blind chat
  preference result, but it does not measure repository completion.

## Uncertainties

- No independent source tested Sol Ultra against Fable Max. Sol Ultra is a
  four-agent product mode; Artificial Analysis tested Sol Max.
- Official plan pages conflict on some Work and Sol Pro eligibility details.
- Exact Fable chat/Cowork effort menus and Cowork web rollout remain
  account-dependent.
- The numeric Grok 4.5 long-context rate above 200K was independently reported
  as double but was not visible in the fetched official table.
- X and Reddit sentiment is mixed and not representative; it is excluded from
  factual conclusions.
