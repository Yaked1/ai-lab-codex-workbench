# Support

This repository provides documentation, prompts, examples, and local validation
scripts. It is not a hosted AI service and cannot grant model access, recover
accounts, change vendor quotas, or provide emergency support.

## Where to ask

- Reproducible bug: open the **Bug report** issue form.
- Small proposed improvement: open the **Feature request** form.
- Bounded coding-agent task: use the **Codex task** form.
- Prompt reliability or safety review: use the **Prompt audit** form.
- Beginner contribution with a named file and check: use the **Good first issue** form.

Before opening an issue, search existing issues and run the relevant local
check. Remove secrets, private paths, tokens, cookies, account identifiers, and
personal documents from logs or screenshots.

## Security and conduct

Use [SECURITY.md](SECURITY.md) for security reports. Use the private maintainer
contact listed in the repository profile for conduct reports. Do not publish a
security vulnerability or personal information in a public issue.

Support is best-effort. A maintainer response, fix, release, or vendor answer is
not guaranteed on a particular schedule.
